# 駅ホーム発車標シミュレータ
v1.1.0準拠（2026/05/15リリース）

## 本プロジェクト概要
本プロジェクトはホーム上で確認できる発車標を模したシミュレータを作成し、終電逃しをなくすことを目的としています。リアルタイム表示対応や実際の時刻表データを表示し、無事に帰宅することに役に立つツールを目指しています（主に自己満足です）。

## 主な機能と使用方法
※v1.1.0にて大規模アップデートを行いました。

### トップページについて
3つの画面を用意しています。画面下部にあるセレクタにて好きなダイヤを選択し、「設定を保存して適用」ボタンを押下してください。設定が保存され、画面表示が切り替わります。

### 初期表示
ページ読み込み時はJR東海道線金山駅の上下線及びJR中央線名古屋駅の平日ダイヤが読み込まれ、次発と次々発がリアルタイムに表示されます。

### ダイヤ選択
ダイヤ選択セレクタから任意のものに切り替えることができます。現在の対応状況はセレクタの中身をご確認ください。

### リアルタイム更新
デフォルトでは動作モードが自動更新となっており、おおよそ1分毎に表示更新が行われます。発車時刻を過ぎた後は通常の発車標と同じく、次発及び次々発が更新されます。  
チェックボックスを外すことで自動更新を止めることが可能です。（※シミュレータページのみ）

### カスタマイズ表示
自動更新機能をオフにし、その下のカスタマイズ入力フォームを使用することで任意の表示をさせることができます。（※シミュレータページのみ）

## 注意事項
- 本プロジェクトに使用している時刻表データは2026年度ダイヤ改正後のものです。2027年度ダイヤ改正後にデータ更新を行うかは未定です。

- 使用している元データは公式ウェブサイトから入手していますが、それをもとに作成したGoogleスプレッドシートの入力データを使用しています。そのため、入力ミスなどによりデータに誤りがある可能性が高いです。あくまでも参考情報として使用するようお願いいたします。

- データ切替時、読み込みまで数秒かかります。お待ちください。

- JR東海様の新型LCD旅客案内装置をもとに作成しているため、種別表示などはJRベースです。今後対応表示を追加するかは検討中です。

- 特急・ホームライナー等の表示は非対応です。ご了承ください。
