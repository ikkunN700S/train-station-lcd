# 駅ホーム発車標シミュレータ
v0.1.5準拠（2026/05/14リリース）

## 本プロジェクト概要
本プロジェクトはホーム上で確認できる発車標を模したシミュレータを作成することを目的としています。それをもとに、リアルタイム表示対応や実際の時刻表データを表示し、何らかの役に立つことを目指しています（主に自己満）。

## 主な機能と使用方法
### 初期表示
ページ読み込み時は名古屋市営地下鉄鶴舞線の塩釜口駅平日下りダイヤが読み込まれ、次発と次々発がリアルタイム表示されます。

### ダイヤ選択
ダイヤ選択ダイアログから対応しているダイヤの内、任意のものに切り替えることができます。現在の対応状況はダイアログの中身をご確認ください。

### リアルタイム更新
デフォルトでは動作モードが自動更新となっており、おおよそ1分毎に表示更新が行われます。発車時刻を過ぎた後は通常の発車標と同じく、次発及び次々発が更新されます。チェックボックスを外すことで自動更新を止めることが可能です。

### カスタマイズ表示
自動更新機能をオフにし、その下のカスタマイズ入力フォームを使用することで任意の表示をさせることができます。

## 注意事項
- 本プロジェクトに使用している時刻表データは2026年度ダイヤ改正後のものです。2027年度ダイヤ改正後にデータ更新を行うかは未定です。

- 使用している元データは公式ウェブサイトから入手していますが、それをもとに作成したGoogleスプレッドシートの入力データを使用しています。そのため、入力ミスなどによりデータに誤りがある可能性が高いです。あくまでも参考情報として使用するようお願いいたします。

- データ切替時、読み込みまで数秒かかります。お待ちください。

- JR東海様の新型LCD旅客案内装置をもとに作成しているため、種別表示などはJRベースです。今後対応表示を追加するかは検討中です。

- 特急・ホームライナー等の表示は非対応です。ご了承ください。
